# Gym Supplement Ordering System

This is a Java-based application developed to demonstrate the use of design patterns, specifically the Factory Method Pattern and Singleton Pattern.

## 📌 Features
- Create different supplement products (Protein, Vitamin, Creatine)
- Dynamic object creation using Factory Method Pattern
- Centralized order management using Singleton Pattern
- Console-based user interaction

## 🛠 Technologies Used
- Java
- NetBeans IDE
- Object-Oriented Programming

## ▶️ How to Run
1. Open the project in NetBeans or any Java IDE
2. Run the MainApp class
3. Enter product type (protein / vitamin / creatine)

## 📂 Design Patterns Used
- Factory Method Pattern
- Singleton Pattern
